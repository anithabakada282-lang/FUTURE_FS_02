# FUTURE_FS_02 - Mini CRM (Client Lead Management System)

## 📌 Description
This is a full-stack Client Lead Management System (Mini CRM) built as part of Future Interns Task 2.

The system allows businesses to manage leads, track their status, and add follow-up notes.

## 🚀 Features
- View all leads
- Update lead status (New → Contacted → Converted)
- Add follow-up notes
- Dashboard with analytics
- Secure login authentication

## 🛠️ Tech Stack
Frontend:
- React.js
- JavaScript
- CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## 📊 Dashboard
- Total leads
- Contacted leads
- Converted leads

## 🔐 Authentication
Token-based authentication using localStorage
